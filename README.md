# Yan Liu — one-page intro

A minimal, single-page personal intro. Mono/terminal aesthetic, light/dark toggle,
hover-driven 3D letter swap on the name, and a masonry video collage.

Open `index.html` in a browser, or serve the folder with any static server:

```bash
python3 -m http.server 8000
# → http://localhost:8000
```

Built with [Claude Code](https://claude.com/claude-code).
